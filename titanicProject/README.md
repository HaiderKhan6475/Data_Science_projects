# Data Science Projects: Titanic & Stock Prediction

This repository contains two main machine learning projects focused on data analysis and predictive modeling.

### 1. Titanic Survival Prediction
- **Goal**: Predict passenger survival outcomes based on features like age, class, and gender.
- **Dataset**: `data/tested.csv`
- **Analysis**: Exploratory Data Analysis (EDA) to find survival trends.

### 2. Stock Market Prediction
- **Goal**: Analyze and predict stock price movements using historical data.
- **Approach**: Time-series analysis and regression-based modeling.

---

### Project Structure
- `notebooks/01_titanic_eda.ipynb` - Titanic Analysis & EDA
- `notebooks/02_stock_prediction.ipynb` - Stock Market Prediction
- `models/titanic_model.pkl` - Trained Machine Learning Model
- `data/` - Dataset files
- `Report/` - Full project documentation

### Installation
To run these notebooks, install the required libraries:
`pip install -r requirements.txt`
